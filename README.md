🌍 Country Info Explorer

A responsive web application that allows users to search and explore information about countries around the world. This project fetches real-time data from a public API and displays details in a clean UI.

🚀 Features
🔎 Search countries by name
🌐 View country details (capital, region, population, flag, etc.)
📱 Fully responsive design
⚡ Fast API-based data fetching
🎨 Styled using Tailwind CSS


🛠️ Tech Stack
HTML
Tailwind CSS
JavaScript
REST API
